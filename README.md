# vue-

项目简介

本项目是基于vue的单页面开发，是模仿淘宝首页制作的，实现了一些界面和基本hover，使用的全是静态数据，旨在练习css和elementUI的使用，是一个纯前端的框架

技术栈

vue-cli +elementUI

如何运行代码

​	1. npm i 安装相应的包

​	2. npm run serve 启动项目
